La substitution mono-alphabétique par mélange

La substitution mono-alphabétique par mélange est une méthode de chiffrement qui consiste à substituer chaque lettres de l'alphabet par une autre lettre choisie aléatoirement. Dans notre cas, ces lettres seront choisies par l'utilisateur.

Prenons un exemple: si l'utlisateur
définit un tableau tel que celui-ci:



La phrase en clair "LA NSI EST MON COURS PREFERE", donnera l'expression
"QKVYA ZYLFE VHEXS ISZPZ SZ"
Vous avez du le remarquer, le message codé se présente par groupe de cinq lettres

⚠️attention: Chaque lettre ne doit n'avoir qu'une seule substitution et inversement, une substitution ne doit être attribué qu'a une seul lettre, sinon le chiffrement ne peut pas marcher.

Maintenant, pour déchiffrer un message codé par substitution mono-alphabétique par mélange, il suffit de faire l'inverse. En reprenant la clé, c'est a dire l'alphabet que l'utilisateur a rentré on peut facilement déchiffrer un mot. Par exemple, si l'on rentre le code "FKFKV" le mot déchiffré est "MAMAN".
