Bienvenue sur ma page web je vais vous présenter la substitution mono-alphabétique par décalage

En cryptographie, la substitution mono-alphabétique par décalage, aussi connu comme le chiffre de César ou le code de César, est une méthode de chiffrement très simple utilisée par Jules César dans ses correspondances secrètes. Le texte chiffré s'obtient en remplaçant chaque lettre du texte clair original par une lettre à distance fixe, toujours du même côté, dans l'ordre de l'alphabet. Pour les dernières lettres (dans le cas d'un décalage à droite), on reprend au début. Par exemple avec un décalage de 3 vers la droite, A est remplacé par D, B devient E, et ainsi jusqu'à W qui devient Z, puis X devient A etc. Il s'agit d'une permutation circulaire de l'alphabet. La longueur du décalage, 3 dans l'exemple évoqué, constitue la clé du chiffrement qu'il suffit de transmettre au destinataire — s'il sait déjà qu'il s'agit d'un chiffrement de César — pour que celui-ci puisse déchiffrer le message. Dans le cas de l'alphabet latin, le chiffre de César n'a que 26 clés possibles.

Décryptage du message

Le décryptage du message chiffré se fait de la même manière que le chiffrement, mais en inversant le décalage. Si le message a été chiffré avec un décalage de 3, il suffira de décaler chaque lettre du texte chiffré de 3 positions vers la gauche dans l'alphabet pour retrouver le message original.
